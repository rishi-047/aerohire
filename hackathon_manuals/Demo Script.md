# Demo Script

This is a recommended demo flow for a 5‑10 minute hackathon presentation.

## 1. Problem Statement

Explain the core pain:
- Traditional hiring assessments are either too shallow or opaque.
- Cheating and resume inflation are hard to detect.
- Recruiters want explainable AI decisions.

## 2. Candidate Flow

Steps:
- Register as candidate
- Upload a resume
- Start the assessment
- Run tests on Q1
- Let the teammate chat pop in Q2 and respond
- End the assessment

Highlight:
- Resume parsing
- Code execution
- Integrity logging
- Culture fit response

## 3. Recruiter Flow

Steps:
- Login as recruiter
- Open candidate detail
- Show:
  - Technical score
  - Code submissions and playback
  - Originality chart
  - Session integrity graph
  - AI audit report

## 4. Glass Box AI

Show AI report:
- Strengths list includes resume skills
- Risks list includes integrity flags
- Summary compares resume vs performance

## 5. Admin Console (Optional)

Show secret admin dashboard:
- Delete candidate
- Re‑register same email

## 6. Closing

Summary:
- End‑to‑end pipeline is live
- AI decisions are explainable
- Integrity system prevents easy cheating

## Tips

Have 1 candidate pre‑seeded for quick demo.
Trigger at least one integrity event to show MED/HIGH in the graph.
